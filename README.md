# 🔐 Password Generator

## Overview
This project is a simple **Password Generator** that creates strong and secure passwords.  
The program randomly generates passwords using a mix of uppercase letters, lowercase letters, numbers, and special symbols to make passwords harder to guess.

---

## Features
- Generates random and secure passwords
- Uses a mix of characters for stronger security:
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
  - Numbers (0–9)
  - Special symbols (e.g., ! @ # $ % & *)
- Each character is randomly selected
- Simple and easy to run

---

## Technologies Used
- Python
- Random library for generating characters
- Basic programming concepts

---

## How It Works
1. The program stores possible characters (letters, numbers, symbols).
2. It randomly selects characters from these groups.
3. The characters are combined to create a secure password.
4. The generated password is displayed to the user.
